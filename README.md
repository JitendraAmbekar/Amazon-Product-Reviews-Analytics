# Amazon Product Reviews Analytics Dashboard

## 📌 Project Overview
An end-to-end data analytics project using Python (Sentiment Analysis), SQL (Data exploration), and Power BI to analyze 42,000+ Amazon product reviews. The goal is to provide actionable insights into product performance, customer satisfaction, and review trends.

## 📊 Dashboard Preview
![Amazon Dashboard](dashboard_screenshot.png) 

## 🛠️ Tech Stack & Tools
* **Data Analytics & Visualization:** Power BI Desktop
* **Programming Language:** Python (for Sentiment Analysis & Data Preprocessing)
* **Database:** SQL

## 💡 Key Insights from the Dashboard
* **Average Rating:** The overall average rating stands at **2.19**, indicating a strong need for product improvements.
* **Sentiment Distribution:** Visualized customer emotions categorized into Positive, Negative, and Neutral.
* **Review Trend:** Tracked how review volumes have changed over the years.

## 🚀 How to Run the Project
1. Clone this repository.
2. Open the `.pbix` file using **Power BI Desktop**.
